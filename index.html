Sistema 3.1

Este é um sistema para calcular o campo elétrico com distribuição contínua de carga sobre um anel carregado.

Sistema 3.2

Esse sistema permite calcular o campo elétrico gerado por um disco carregado eletricamente. Os usuários inserem informações como raio, posição e carga do disco, e o sistema realiza automaticamente os cálculos, exibindo o resultado na página de forma acessível e visual.

Sistema 3.3

O sistema realiza cálculos relacionados a um campo elétrico, apresenta as diferentes partes da fórmula e os resultados finitos e infinitos, utilizando notação científica.
